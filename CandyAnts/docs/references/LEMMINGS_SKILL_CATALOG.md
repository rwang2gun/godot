# 레밍즈 스킬 카탈로그 — CandyAnts 확장 후보 조사

**작성 일자**: 2026-05-08
**목적**: CandyAnts MVP 8종 외에 추가할 수 있는 스킬·캐릭터 아이디어 발굴
**출처**: Lemmings(1991), Oh No! More Lemmings, Lemmings 2: The Tribes, Lemmings 3D, Lemmings Revolution, Lemmings(2006 PSP), Lemmings Touch, 기타 공식·팬 확장

---

## 0. 분류 축

각 스킬을 다음 3축으로 정리:

1. **원작 동작** — 레밍즈에서 어떻게 동작하는지
2. **CandyAnts 각색안** — 사탕 왕복 운반 + 사이드뷰 개미 컨셉으로 옮길 때
3. **구현 난이도** — 현 아키텍처 기준 추정

### 구현 난이도 기준
| 기호 | 난이도 | 의미 |
|---|---|---|
| 🟢 | Easy | 기존 시스템 확장 (StateMachine 추가 또는 기존 상태 변형) |
| 🟡 | Medium | 새 메카닉 추가 (새 물리 상호작용 / 새 데이터) |
| 🔴 | Hard | 신규 코어 시스템 필요 (AI 경로 / 신규 엔티티 / 새 충돌 레이어) |
| ⚫ | Very Hard | 아키텍처 결정 변경 필요 (ADR 추가/수정) |

### CandyAnts 적합도 기준
| 기호 | 의미 |
|---|---|
| ⭐⭐⭐ | 사탕 운반 + 사이드뷰 개미 컨셉과 자연스럽게 맞물림 |
| ⭐⭐ | 컨셉 각색하면 매력적, 일부 메카닉 재해석 필요 |
| ⭐ | 컨셉상 거리 있음, 굳이 넣으려면 큰 재해석 필요 |

## 1. 원작 8종 (이미 MVP 계획에 포함됨)

> 참고용. CandyAnts MVP Phase 2~11에서 단계적으로 구현 예정.

| 스킬 | 원작 동작 | CandyAnts 각색 (현재 PRD 기준) | 구현 |
|---|---|---|---|
| **Climber** | 벽 수직 등반 | 동일 — 운반 중에도 등반 가능 (Carrying 0.78배 속도 유지) | 🟢 |
| **Floater** | 낙하산 펼치고 천천히 낙하 | 풍선 또는 잎으로 낙하 속도 감소. Carrying 시 사탕 무게로 효과 약화 변형 가능 | 🟢 |
| **Bomber** | 5초 카운트다운 후 자폭, 주변 지형 원형 파괴 | 동일 — 사탕 운반 중 자폭 시 사탕 영구 손실(lost +1) | 🟡 |
| **Blocker** | 좌우 막아 다른 레밍 회전시킴 | 동일 — 다른 개미 회전. Blocker 자신은 운반 불가 (배타적 상태) | 🟢 |
| **Builder** | 12칸 계단 쌓기 | 동일 — 잎/사탕 부스러기로 발판 형성 | 🟡 |
| **Basher** | 수평 굴착 | 동일 — TileMap 동적 파괴 | 🔴 |
| **Miner** | 대각선 아래 굴착 | 동일 | 🔴 |
| **Digger** | 수직 아래 굴착 | 동일 | 🔴 |

## 4. CandyAnts 추천 추가 후보 (필터링 결과)

### Tier 1 — 강력 추천 (적합도 ⭐⭐⭐ + 구현 🟢/🟡)

이 4개는 **컨셉 적합도 + 구현 난이도** 양면 모두 우수. MVP 후 v1.1~v1.2 콘텐츠 확장 시 우선 검토.

| 스킬 | 적합도 | 구현 | 추가 가치 |
|---|---|---|---|
| **Walker** (un-blocker) | ⭐⭐⭐ | 🟢 | Blocker가 배타적 상태(운반 불가)인 CandyAnts에서 자원 회수 의미 큼 |
| **Stacker** (수직 쌓기) | ⭐⭐⭐ | 🟡 | Builder가 계단이면 Stacker는 좁은 수직 통로 — 레벨 디자인 다양성 |
| **Stomper** (충격 파괴) | ⭐⭐⭐ | 🟡 | Bomber는 자살, Stomper는 비자살 + 약함 → 운반 중 사용 가능 |
| **Disarmer** (Hazard 비활성) | ⭐⭐⭐ | 🟡 | Hazard 시스템(Phase 5)과 자연스럽게 맞물림. 라우팅 다양성 ↑ |

### Tier 2 — 검토할 만함 (적합도 ⭐⭐⭐ + 구현 🔴, 또는 ⭐⭐ + 🟢/🟡)

확장팩 컨셉에 따라 추가 고려.

| 스킬 | 적합도 | 구현 | 비고 |
|---|---|---|---|
| **Surfer/Swimmer** | ⭐⭐⭐ | 🟡 | Water Hazard 통과 메카닉. Lifeguard와 묶어서 "수상 부족" 콘셉트 가능 |
| **Glider** (수평 비행) | ⭐⭐⭐ | 🟡 | Floater의 수평 변형. 갭 큰 레벨에서 유용 |
| **Platformer** (수평 발판) | ⭐⭐⭐ | 🟡 | Builder는 계단, Platformer는 평면. 좁은 갭 처리 |
| **Acrobat/Jumper** (점프) | ⭐⭐⭐ | 🟡 | 작은 갭 즉시 도약 |
| **Throw** (개미 던지기) | ⭐⭐⭐ | 🔴 | 그룹 협력 — 사탕 운반 함의 깊음. 신규 메카닉 |
| **Lifeguard** (수중 구조) | ⭐⭐⭐ | 🔴 | MVP에서 제외했던 "운반자 사망 시 사탕 회수" 부활. Score 시스템 4-카운터에 자연스럽게 맞물림 |
| **Slider** (벽 미끄러짐) | ⭐⭐ | 🟢 | Climber의 하강 버전 — 가벼운 추가 |
| **Chiller** (Water Hazard 동결) | ⭐⭐⭐ | 🟡 | Water Hazard 임시 비활성화. 대안 라우팅 |

### Tier 3 — 컨셉 확장 시 (적합도 ⭐⭐⭐ + 구현 ⚫, 또는 ⭐⭐)

새 ADR이 필요하거나 기존 결정과 충돌. **CandyAnts 정체성 변경 동반**.

| 스킬 | 적합도 | 구현 | 검토 사항 |
|---|---|---|---|
| **Cloner** | ⭐⭐ | ⚫ | 사탕 HP 자원 시스템과 충돌 — 복제된 개미는 새 HP 부담 X 등의 규칙 필요 |
| **Magnet** | ⭐⭐ | 🔴 | 페로몬 시스템과 결합 시 흥미로움. 단 ADR-005(단순 180° 귀환)와 충돌 — 페로몬 도입 동반 검토 |
| **Genie / 메타 스킬** | ⭐ | 🔴 | 글로벌 효과 — Release Rate 슬라이더와 같은 카테고리 |

## 5. 부족(테마) 단위 콘텐츠 확장 아이디어

CandyAnts는 사탕 운반 컨셉이 강함. 부족 다양화로 풀 수 있는 테마:

| 테마 | 스킬 묶음 | 컨셉 |
|---|---|---|
| **수상 개미** | Surfer + Swimmer + Lifeguard + Chiller | Water Hazard 중심 부족. 시럽·꿀 환경 |
| **공중 개미** | Glider + Acrobat + Jumper + Floater 강화 | 갭과 고도가 많은 레벨. 잎/풍선 활용 |
| **건축 개미** | Stacker + Platformer + Builder + Stoner | 지형 형성 특화. 복잡한 미로 레벨 |
| **파괴 개미** | Stomper + Basher + Miner + Digger + Disarmer | 장애물 제거 특화. 격렬한 레벨 |
| **협력 개미** | Throw + Walker + Magnet + Lifeguard | 그룹 동작 특화. 군중 관리가 핵심 레벨 |

→ **5개 부족 × 4~5개 스킬 = 20~25개 신규 스킬**. 본 MVP 8개 외에 v2.0 정도의 대규모 확장.

---

## 6. 우선 검토 추천 — v1.1 / v1.2 후보 5개

MVP 완료 후 가장 먼저 추가하면 효과 좋을 것 추천:

1. **Walker** — Blocker 해제. 자원 회수 메카닉. 구현 가장 가벼움
2. **Stacker** — 수직 발판. 레벨 디자인 다양성 큼. Builder 코드 80% 재사용
3. **Stomper** — 비자살 파괴. 운반 중에도 사용 가능 → 전략 다양성
4. **Disarmer** — Hazard 영구 비활성. Phase 5 Hazard 시스템 직접 활용
5. **Glider** — 수평 비행. Floater의 자연스러운 진화

**한 번에 다 넣지 말고 1~2개씩 추가 → 메타 균형 확인 → 다음** 권장. 한꺼번에 5개 넣으면 기존 8종 가치 희석됨 (콘텐츠 기획 §3 Pillar 6 "직업 자원 분배의 묘미"가 결정 부담 늘면서 무뎌짐).

## 7. 검토 시 주의 사항

### 7.1 ADR 충돌 점검
신규 스킬 추가 시 다음 ADR 영향 검토 필수:

| ADR | 영향 가능 스킬 | 검토 포인트 |
|---|---|---|
| ADR-002 (사탕 4-카운터) | Cloner, Lifeguard | 새 카운터 추가 또는 기존 카운터 의미 변경 |
| ADR-003 (SkillRegistry preload) | 모든 신규 | preload 1줄 추가 잊지 말 것 (자기등록 X) |
| ADR-005 (180° 귀환) | Magnet | 페로몬 도입 시 ADR 새로 필요 |
| ADR-006 (Carrying 0.78배) | Stomper, Glider | 운반 중 사용 가능한지 — 명시적으로 결정 필요 |

### 7.2 Pillar 균형 점검
콘텐츠 기획서(레밍즈 클론 §3) 7개 Fun Pillar 중 어떤 걸 강화/약화하는지 점검:

| 신규 스킬 | Pillar 영향 |
|---|---|
| Walker | Pillar 6 (자원 분배) ↑ — Blocker 회수 가능 |
| Stacker | Pillar 1 (퍼즐 풀이) ↑ — 새 솔루션 경로 |
| Stomper | Pillar 2 (압박감) ↑ — 즉시 사용 가능한 파괴 |
| Disarmer | Pillar 7 (창의적 풀이) ↑ — 우회 vs 무력화 선택지 |
| Glider | Pillar 4 (군중 관리) ↑ — 공중 라우팅 추가 |
| Cloner | Pillar 6 (자원 분배) ↓↑ — 균형 깨짐 위험 |

### 7.3 모바일 이식 영향
조작 정밀도 요구가 높은 스킬은 모바일 이식 시 추가 작업:

| 스킬 | 모바일 우려 |
|---|---|
| Throw | 정확한 방향·세기 입력 필요 |
| Magnet | 영향 범위 시각화 필요 |
| Tightrope Walker | 좁은 발판 클릭 정확도 |
| Stilts | 키 변경 시 충돌 박스 갱신 |
